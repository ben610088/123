# NDI TVC F-16 MATLAB 模型

## 简介

这是一个基于 MATLAB 的 F-16 飞机推力矢量控制（TVC）模型。该项目的目的是模拟和控制 F-16 飞机的推力矢量，以实现更高效的机动性和更精确的飞行控制。通过实现推力矢量控制（TVC）算法，本项目可以在 F-16 模型中应用垂直和水平推力矢量的动态控制，模拟其在各种飞行条件下的表现。
