# NDI TVC F-16 MATLAB Model

## 簡介

這是一個基於 MATLAB 的 F-16 飛機推力向量控制（TVC）模型。該專案的目的是模擬和控制 F-16 飛機的推力向量，以實現更有效率的機動性和更精確的飛行控制。透過實現推力向量控制（TVC）演算法，本項目可在 F-16 模型中應用垂直和水平推力向量的動態控制，模擬其在各種飛行條件下的表現。
